Quick Start Guide

How to use the API

Authentication

Include your API key in the request header:

X-API-Key: your_api_key_here

Example Request

curl -X GET {{ request.url_root }}api/v1/vps \
  -H "X-API-Key: your_api_key_here" \
  -H "Content-Type: application/json"

Available Endpoints

GET /api/v1/vps
List all VPS
GET /api/v1/vps/:id
Get VPS details
POST /api/v1/vps/:id/start
Start VPS
POST /api/v1/vps/:id/stop
Stop VPS
GET /api/v1/users
List all users (admin only)
GET /api/v1/nodes
List all nodes (admin only)
